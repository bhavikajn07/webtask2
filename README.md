## About

A cyberpunk-themed AI landing page featuring animated particle backgrounds, glassmorphism cards, scroll-triggered reveals, glitch text effects, and a working contact form with validation.

---

## Sections

| Section | Features |
|---------|----------|
| **Hero** | Particle canvas, glitch text, stats counters, CTAs |
| **Features** | 6 glassmorphism cards with hover tilt |
| **About** | Holographic helmet image, tech tags, stats |
| **Testimonials** | 3 user review cards with star ratings |
| **Contact** | Working form with validation + notifications |
| **Footer** | Navigation links and social icons |

---

## Tech Stack

- **HTML5** — Semantic markup with ARIA labels
- **CSS3** — Custom properties, glassmorphism, gradients, animations
- **JavaScript** — Vanilla JS, IntersectionObserver, Canvas API
- **Fonts** — Orbitron, Space Grotesk, Inter (Google Fonts)
- **Icons** — Font Awesome 6.5
